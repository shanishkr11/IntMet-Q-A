Week 2: Web API, Security & Azure Deep Dive

Architecture 1. How do you implement "Content Negotiation" in ASP.NET Core Web API?
Content negotiation occurs when the client specifies the format of the data it wants (JSON, XML) using the Accept header. .NET Core handles JSON by default but can be configured to support XML or custom formatters in Program.cs.
Real-time Example: In the RAB application, we ensured the API could return XML for legacy system integrations while providing JSON for our modern Angular frontend[cite: 1].
Performance 2. What is "Eager Loading" vs. "Lazy Loading" in Entity Framework Core?
Eager Loading uses .Include() to fetch related data in a single SQL join. Lazy Loading fetches related data only when it is accessed, which can lead to the "N+1" problem.
Real-time Example: In the E-commerce project, I used Eager Loading for Order details to minimize database roundtrips, significantly improving page load speed[cite: 1].
Security 3. How do you handle Refresh Tokens in JWT Authentication?
Since JWTs are short-lived for security, I use a Refresh Token stored in a secure, HTTP-only cookie. When the JWT expires, the client sends the refresh token to a specific endpoint to obtain a new JWT without re-logging.
Real-time Example: I implemented this in the E-commerce admin panel to provide a seamless user experience while maintaining high security standards[cite: 1].
Azure 4. What is the difference between Azure App Service and Azure Functions?
App Service is for hosting full web applications (Always On). Azure Functions are serverless, event-driven, and scale automatically based on triggers (Timer, HTTP, Blob).
Real-time Example: For the ELOGR project, I could have used Azure Functions to trigger the Open XML document generation whenever a new data entry was submitted[cite: 1].
Database 5. When would you use Azure Cosmos DB over Azure SQL?
I use Cosmos DB for unstructured/semi-structured data requiring global scale and low latency. I use Azure SQL for relational data with complex transactions (ACID compliance).
Real-time Example: As a certified Azure Developer (AZ-204), I understand how to choose Cosmos DB for flexible schema needs like product catalogs in global applications[cite: 1].
Web API 6. How do you handle API Versioning?
API versioning can be done via URL (/api/v1/), Query String, or Request Headers. This allows us to push breaking changes without affecting existing clients.
Real-time Example: I used URL-based versioning at Cognine to ensure that third-party integrations didn't break when we updated the RAB application modules[cite: 1].
Security 7. What is CORS and why is it important in Full-Stack development?
Cross-Origin Resource Sharing (CORS) is a security feature that restricts web pages from making requests to a different domain than the one that served the page. We must configure it in .NET to allow our Angular frontend to talk to the Web API.
Real-time Example: I configured CORS policies in my .NET Core Web API to only allow requests from our specific production and staging Angular domains[cite: 1].
C# / .NET 8. What are "Action Filters" in ASP.NET Core?
Action filters allow you to run code before or after an action method executes. They are useful for logging, validation (ModelState), or caching.
Real-time Example: I created a custom validation filter to automatically check ModelState.IsValid across all my controllers, reducing boilerplate code[cite: 1].
Azure 9. How does Azure Blob Storage help with application scalability?
By offloading static files (images, PDFs) to Blob Storage, the web server focuses purely on logic. It also provides features like CDN integration for faster global delivery.
Real-time Example: In the ELOGR project, storing the generated Word documents in Azure Blob Storage would have been more efficient than storing them on the local server[cite: 1].
Database 10. Explain the use of "Automapper" in .NET projects.
AutoMapper is used to map data from one object type (like a Database Entity) to another (like a DTO - Data Transfer Object) automatically, keeping the code clean.
Real-time Example: I used AutoMapper to hide sensitive database fields from being exposed to the Angular frontend via DTOs[cite: 1].
Cloud Architecture 11. What is a "Service Bus" in Azure?
Azure Service Bus is a highly reliable cloud messaging service between applications. It helps in decoupling microservices by using queues and topics.
Real-time Example: My focus on microservices communication includes using message brokers like Service Bus to ensure reliable data transfer between bounded contexts[cite: 1].
Testing 12. What is your approach to Unit Testing for Web APIs?
I use xUnit or NUnit along with Moq. I mock the dependencies (like the database or external services) to test the controller logic in isolation.
Real-time Example: I developed and executed unit tests for web components at Cognine to ensure high-quality releases and reduce production bugs[cite: 1].
